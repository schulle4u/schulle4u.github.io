My repositories

Plugins for Datenstrom Yellow

  • Audio: A basic HTML5 audio player for Yellow CMS
  • Akismet: An attempt to integrate akismet service into Yellow. This is not finished yet.
  • Global content: Includes page content into other pages. Useful to e.g. create a global sidebar.
  • Message of the day: Message of the day plugin that uses page metadata to display simple text messages for each weekday.
  • Podcast: Podcast feed plugin for Datenstrom Yellow
  • RadioBoss: Shortcut plugin to integrate the widgets for RadioBoss Cloud, a cloud-based radio automation service by DJSoft.net currently in development.
  • Random: Plugin to display random pages from specified location.
  • Shariff: Privacy-friendly social sharing buttons for Yellow.
  • Skipto: Yellow plugin that adds SkipTo-links for keyboard navigation.
  • Ticker: A RSS parser plugin for yellow.
  • TuneIn: Shortcut plugin to embed TuneIn Radio player.

Miscellaneous

  • Streamkick: Shoutcast and icecast stream kick. Code from my attic, no longer maintained.

Other projects

  • Powerradio4u (german): Internet radio station for content available under a creative commons license.
